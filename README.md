# Simple E-commerce Store

## Features
- Product Listing
- Product Details Page
- Shopping Cart
- User Registration/Login
- Order Processing
- MongoDB Database

## Install Steps

### 1. Install Node Modules
npm install

### 2. Start MongoDB
Make sure MongoDB is running locally.

### 3. Run Server
node server.js

### 4. Open Browser
http://localhost:3000

### 5. Seed Products
http://localhost:3000/seed
